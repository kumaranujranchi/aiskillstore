# Execoore Fintech Theme Design System

This skill provides the design tokens and rules for the Execoore Fintech theme. Use these specifications when generating code to ensure consistency with the theme's modern, tech-focused aesthetic.

## 1. Color Palette

- **Primary Blue**: `#0056b3` - The core brand color, used for primary buttons, active states, and key highlights. Represents trust and stability.
- **Background White**: `#ffffff` - The main background color, ensuring a clean and spacious feel.
- **Dark Text**: `#1e2125` - High-contrast color for headings and primary content.
- **Secondary Text**: `#6c757d` - Used for subtitles, descriptions, and less critical information.
- **Light Gray Background**: `#f8f9fa` - Used for secondary sections, cards, and subtle contrasts.

## 2. Typography

- **Headings**:
  - **Font Family**: `Inter`, sans-serif (Modern, clean, highly legible).
  - **Weight**: 600 (Semi-Bold) to 700 (Bold).
  - **Style**: Function-first, geometric but approachable. Used for all section titles and hero text.

- **Body Text**:
  - **Font Family**: `Roboto`, sans-serif (Neutral, versatile).
  - **Weight**: 400 (Regular).
  - **Size**: `16px` base.
  - **Line Height**: `1.6` for comfortable reading.

## 3. Component Styles

### Buttons
- **Primary Button**:
  - Background: `#0056b3`
  - Text: `#ffffff`
  - Border Radius: `4px` (Slightly rounded, keeping it professional but modern).
  - Padding: `12px 30px`.
  - Hover: Darken to `#004494`.
  - Transition: `0.3s ease`.

- **Secondary/Outline Button**:
  - Background: Transparent.
  - Border: `2px solid #0056b3`.
  - Text: `#0056b3`.
  - Border Radius: `4px`.
  - Hover: Background fills with `#0056b3`, text becomes `#ffffff`.

### Cards
- **Background**: `#ffffff`.
- **Border**: `1px solid #e9ecef`.
- **Shadow**: `0 0.5rem 1rem rgba(0, 0, 0, 0.15)` (Soft but distinct elevation).
- **Border Radius**: `8px`.
- **Padding**: `2rem` (Generous spacing).

### Navbar
- **Background**: White or Transparent (on hero sections).
- **Links**: Dark text `#1e2125`, turning Primary Blue `#0056b3` on hover.
- **Sticky**: Often becomes white with a shadow on scroll.

### Footer
- **Background**: Dark (`#1e2125` or similar dark charcoal).
- **Text**: White or Light Gray (`#adb5bd`).
- **Links**: White, turning Blue on hover.

## 4. Layout & Spacing
- **Container Width**: Standard bootstrap-like container (approx `1140px` or `1200px`).
- **Section Spacing**: `80px` to `100px` vertical padding.
- **Grid System**: Based on a 12-column flexible grid.

## 5. Design Philosophy
Execoore is built for "Performance and Professionalism." It avoids unnecessary clutter, focusing on clear typography, strong calls to action, and a color palette that inspires confidence (Blue/White/Gray). It is ideal for startups, crypto exchanges, consulting firms, and SaaS platforms.
